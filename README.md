# Lifestyle Connect

Connection at your own pace. A platform for intentional ENM couples.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deploy to Vercel:

```bash
vercel
```

## Features

- Multi-step profile creation
- Topic lounges with anonymous participation
- Local events and coordination
- Places & presence awareness layer
- Dual-partner confirmation gates
